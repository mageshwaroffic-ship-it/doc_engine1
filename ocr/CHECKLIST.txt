🔌 API ENDPOINTS INTEGRATED
══════════════════════════════════════════════════════════════════

Endpoint: POST /extract
├─ Input: Image file (multipart)
├─ Processing:
│  ├─ Image preprocessing
│  ├─ OCR extraction
│  └─ Text cleaning
├─ Output:
│  ├─ job_id (for reference)
│  ├─ extracted_text (OCR result)
│  ├─ languages_used (kan, eng)
│  └─ searchable_pdf (URL reference)
└─ Used by: "Extract Text" button → Frontend displays text boxes

Endpoint: POST /formatted
├─ Input: Image file (multipart)
├─ Processing:
│  ├─ Run OCR on image
│  ├─ Get bounding boxes for text
│  ├─ Create DOCX with background
│  └─ Overlay text boxes at exact positions
├─ Output: DOCX file (binary)
└─ Used by: "Download DOCX" button

Endpoint: GET /download/{job_id}
├─ Input: job_id (from /extract response)
├─ Processing: Retrieve pre-generated PDF from ocr_workspace
├─ Output: Searchable PDF file (binary)
└─ Used by: "Download PDF" button
